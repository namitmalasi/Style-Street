# StyleStreet — Full Stack MERN Ecommerce Application

StyleStreet is a modern full-stack ecommerce web application built using the MERN stack.  
The application provides a complete online shopping experience with secure authentication, product browsing, cart management, and Stripe payment integration.

---

# Features

## Authentication

- JWT-based authentication
- Secure HTTP-only cookie storage
- Protected routes
- Persistent login sessions

## Ecommerce Functionality

- Product listing page
- Dynamic product details page
- Shopping cart functionality
- Quantity management
- Order history
- Responsive UI

## Payments

- Stripe Checkout integration
- Secure online payments
- Success and cancel payment flow

## State Management

- Zustand for global state management
- Persistent cart and authentication state

## UI/UX

- Responsive modern design
- TailwindCSS styling
- Toast notifications
- Optimized image loading
- Lazy loading images
